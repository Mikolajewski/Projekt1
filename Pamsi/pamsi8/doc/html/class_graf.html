template<class krawedz, class wierzcholek>

Klasa Graf.

Klasa zawierajaca strukture grafu. W sklad klasy wchodza metody wykonujace operacje na grafie m.in. przeszukania grafu. Graf jest oparty na tablicy z hashowaniem.
